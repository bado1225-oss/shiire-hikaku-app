# 仕入れ比較アプリ

飲食店の仕入れ商品について、現在の仕入先（旧）と相見積もり後の新仕入先を比較し、月間/年間の経費削減額を可視化する Web アプリです。

## 主な機能

- **比較入力**: 商品ごとに「旧仕入先」「新仕入先」の単価・送料・発注頻度を登録
- **自動計算**: 月間商品代・送料・合計、月間/年間削減額、削減率
- **ランキング**: 削減額の大きい順、店舗別合計
- **試算**: 1商品に対して複数候補の見積もりを比較し、BESTを提案
- **Excelインポート**: 既存の「仕入れ比較シート.xlsx」から一括取り込み
- **CSV/JSONエクスポート**: バックアップ・他ツール連携
- **PWA対応**: スマホのホーム画面に追加して、アプリ感覚で使える

## 構成

- 静的 HTML + CSS + JavaScript（フレームワーク不要）
- データ保存: ブラウザの localStorage（端末内のみ・クラウド同期なし）
- 仕入れ比較シート読み込み: [SheetJS](https://sheetjs.com/)（CDN）
